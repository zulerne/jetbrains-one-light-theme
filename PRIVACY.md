# Privacy Policy

**Last updated: January 24, 2026**

The "One Light Theme" plugin is a UI theme for JetBrains IDEs.

## Data Collection

This plugin is a pure theme (color scheme and UI adjustments). It does not collect, store, or transmit any personal data, usage statistics, or analytics from your machine.
